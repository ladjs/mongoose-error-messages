## Install

[npm][]:

```sh
npm install @ladjs/mongoose-error-messages
```

[yarn][]:

```sh
yarn add @ladjs/mongoose-error-messages
```


## Usage

```js
const mongoose = require('mongoose');
mongoose.Error.messages = require('@ladjs/mongoose-error-messages');
```
